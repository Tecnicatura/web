Formato de archivo

Formato de archivos

  • Texto Plano: contiene bytes solo de caracteres legibles
    • Textos, Archivos de Configuración, Comunicaciones, ...
  • Binario: contiene bytes en todo el rango
    • Imagen, Ejecutable, Audio, Sistema, Video, Comunicación gran cantidad de datos,etc..

Ejemplos de Formatos para datos

  • JSON
  • CSV
  • XML/HTML

JSON

{"nombre":"Juan", "apellido":"Perez", "direccion":"Roca 150"}

JSON array

{[ {"nombre":"Juan", "apellido":"Perez", "direccion":"Roca 150"}, {"nombre":"Pedro", "apellido":"Rodriguez", "direccion":"Alvear 200"}]}

JSON anidado

{"Cliente":{"nombre":"Juan", "apellido":"Perez", "direccion":"Roca 150"}}

CSV

nombre;apellido;direccion Juan;Perez;Roca 150 Pedro;Rodriguez;Alvear 200

HTML

<HTML> <DIV> </DIV> </HTML>
